O sistema deve criar um rascunho na categoria Política, associar tags, subir uma imagem destacada e inserir uma imagem no corpo da matéria.

Este conteúdo não deve ser publicado no site. Ele deve permanecer como rascunho para conferência técnica.

O teste também serve para validar o caminho completo entre ChatGPT, GitHub Actions e WordPress.
